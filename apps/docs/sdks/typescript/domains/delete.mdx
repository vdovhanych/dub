---
title: "Delete"
"og:title": "dub.domains.delete - Delete a domain - TypeScript SDK"
description: "Delete a domain from a workspace. It cannot be undone. This will also delete all the links associated with the domain."
---

## Arguments

<ParamField body="slug" type="string" required>
  Name of the domain. Eg. `acme.com`.
</ParamField>

## Response

<ResponseField name="slug" type="string" required>
  Name of the domain that was deleted.
</ResponseField>

<RequestExample>

```ts
await dub.domains.delete("acme.com");
```

</RequestExample>

<ResponseExample>

```ts
{
  slug: "acme.com";
}
```

</ResponseExample>
